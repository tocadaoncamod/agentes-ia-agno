# Pesquisa Web Profissional

## Quando Usar
- Usuário pede informações atuais ou recentes
- Perguntas sobre notícias, eventos, tendências
- Qualquer tema que exija dados atualizados

## Fluxo de Trabalho

### 1. Entender a Pergunta
- Identifique o tema central e subtópicos
- Determine se precisa de dados atuais (use busca) ou conhecimento geral
- Planeje 2-3 buscas diferentes para cobrir ângulos distintos

### 2. Pesquisar com Múltiplas Fontes
- Use **TavilyTools** como fonte primária (resultados de alta qualidade)
- Use **DuckDuckGoTools** como fallback ou segunda opinião
- Faça buscas em português E inglês quando relevante
- Busque pelo menos 2 fontes diferentes para validação cruzada

### 3. Avaliar e Filtrar
- Priorize fontes oficiais, acadêmicas e jornalísticas
- Descarte informações sem fonte clara
- Identifique viés ou opiniões disfarçadas de fatos
- Verifique datas — informações antigas podem estar desatualizadas

### 4. Apresentar Resultados

#### Formato Padrão de Resposta:
```
## 📊 [Título do Tema]

### Resumo
[2-3 frases com a resposta direta]

### Detalhes
- **Ponto 1**: [explicação] *(Fonte: [nome])*
- **Ponto 2**: [explicação] *(Fonte: [nome])*
- **Ponto 3**: [explicação] *(Fonte: [nome])*

### Contexto Adicional
[Informações de background relevantes]

### Fontes
1. [Nome da fonte] — [URL]
2. [Nome da fonte] — [URL]
```

## Regras Importantes
- **SEMPRE cite fontes** — nunca apresente dados sem referência
- **Seja transparente** — se não encontrou informação, diga claramente
- **Data matters** — sempre mencione quando a informação foi publicada
- **Português brasileiro** — responda sempre em PT-BR
- **Organize com tópicos** — nunca despeje texto corrido
